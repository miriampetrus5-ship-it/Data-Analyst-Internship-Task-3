# Data-Analyst-Internship-Task-3
An interactive dashboard developed as a part of a Data Analys Internship. The project analyzes sales performance using KPIs, trend analysis, regional comparisons and customer segmentation to support business decision-making
## Sales Performance dashboard
## Project Overview
This project was completed as a part of a Data Analyst Internship (Task 3). The objective was to design an interactive Power Bi dashboard that enables business stakeholders to monitor sales performance and make informed decisions using data visualization.
## Objectives
- Develop an interative sales dashboard in Power BI.
- Present key business metrics using KPI cards
- Analyze sale trends overtime
- Support business decion-making through data visualization.
## Dataset
- Kaggle Sales dataset
- Features include:
- - Order date
  - Sales
  - Profit
  - Quantity
  - Discount
  - Region
  - Segment
  - Category
  - Sub-Category
  - Product name
## Dashboard Features
The dashboard includes
- Total sales KPI
- Total profit KPI
- Total Orders KPI
- Total quantity sold KPI
- Average discount KPI
- Sales trend analysis
- Sales by region
- Sales by sub-category
- Profit by segment
## Key insights
- Slaes showed consistent growth throughout the analysis period.
- The west region generated the highest overall profit.
- Chairs recorded the highest sales among the available sub-categories
- The corporate segment contributed significantly to total profit
- The average discount offered across orders was 17%
